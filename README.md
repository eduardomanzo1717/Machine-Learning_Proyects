# Machine Learning Projects

Repositorio que contiene proyectos desarrollados durante la Licenciatura en Matemáticas enfocados en aprendizaje automático utilizando Python y Scikit-Learn.

## Objetivo

Aplicar técnicas de Machine Learning para resolver problemas de clasificación y agrupamiento de datos, reforzando conocimientos de análisis de datos, modelado y visualización.

---

## Tecnologías Utilizadas

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn
- Jupyter Notebook

---

## Proyectos

### K-Means Clustering

Implementación del algoritmo de agrupamiento K-Means para identificar patrones y segmentar observaciones en grupos homogéneos.

**Habilidades demostradas:**
- Aprendizaje no supervisado
- Clustering
- Preparación de datos
- Visualización de resultados
- Evaluación de agrupamientos

**Herramientas:**
- Python
- NumPy
- Matplotlib
- Scikit-Learn

---

### Support Vector Machine (SVM)

Implementación de un modelo de clasificación utilizando Support Vector Machines (SVM) para separar observaciones mediante hiperplanos óptimos.

**Habilidades demostradas:**
- Aprendizaje supervisado
- Clasificación
- Entrenamiento de modelos
- Evaluación de desempeño
- Visualización de fronteras de decisión

**Herramientas:**
- Python
- NumPy
- Matplotlib
- Scikit-Learn

---
